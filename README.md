# AI Mobile Edge Chat (Gemma 4, Offline)
A privacy-first mobile chat application that runs a local LLM entirely on-device using Gemma 4. No API keys, no network calls, no data leaving the user’s phone. Designed for Android and iOS with a clean architecture that you can extend for real products.


### Why this project Most “AI apps” depend on cloud APIs. This project demonstrates a different path:

- True on-device AI with Gemma
- Offline-first experience (works without internet)
- Low-latency responses with hardware acceleration
- User privacy by design (no data collection required)
